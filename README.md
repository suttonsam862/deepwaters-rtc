# Deep Waters RTC — Coaching Ledger

Private session & payment tracker for Deep Waters RTC, built on Shopify Hydrogen
(hosted free on Oxygen) with a Supabase Postgres backend. The app is at `/tracker`.

➡️ **See [SETUP.md](./SETUP.md) for full setup & deploy instructions.**

Quick start:
```bash
npm install
cp .env.example .env   # add your Supabase URL + anon key
npm run dev            # open the printed URL + /tracker
```
